# Kebijakan Privasi Amalanku

_Terakhir diperbarui: [17 Agustus 2025]_

Amalanku ("Aplikasi") dirancang untuk membantu umat Muslim menjaga konsistensi ibadah harian. Kebijakan ini menjelaskan data apa yang dikumpulkan dan bagaimana kami menanganinya dengan menghormati privasi Anda.

## ğŸ” Pengumpulan Data

### Data yang Dikumpulkan:

- **Data Minimal**: Amalanku dirancang untuk mengumpulkan data sesedikit mungkin dan sebagian besar beroperasi secara offline  
- **Catatan Amalan**: Rekaman amalan harian Anda (shalat, dzikir, membaca Al-Quran) tersimpan **hanya di perangkat Anda**  
- **Preferensi Pengguna**: Pengaturan notifikasi, reminder, dan konfigurasi aplikasi (disimpan secara lokal)  
- **Waktu Shalat & Masjid Terdekat**: Perhitungan waktu shalat dan lokasi masjid terdekat berdasarkan lokasi, diproses secara lokal  

### Data yang **Tidak** Dikumpulkan:

- **Informasi Pribadi**: Tidak mengumpulkan nama, email, nomor telepon, atau data identitas lainnya  
- **Analitik Penggunaan**: Tidak memantau cara Anda menggunakan aplikasi  
- **Informasi Perangkat**: Hanya sesuai kebutuhan fungsi aplikasi  
- **Data Akun**: Tidak ada akun online atau cloud, semua tetap di perangkat  
- **Data Perilaku**: Tidak memantau atau menganalisis aktivitas spiritual Anda  

### Penyimpanan Lokal:

- **Tujuan**: Menyimpan catatan amalan, preferensi, dan pengaturan aplikasi secara lokal  
- **Lokasi**: Semua data tersimpan **hanya di perangkat Anda**  
- **Kontrol**: Anda dapat menghapus data kapan saja melalui pengaturan aplikasi  

## ğŸŒ Izin & Akses Jaringan

### Izin yang Diperlukan:

- **Storage**: Untuk mengekspor catatan amalan ke PDF sebagai cadangan atau dibagikan  
- **Notifikasi**: Untuk reminder waktu shalat dan amalan  
- **Lokasi (Opsional)**: Untuk menghitung waktu shalat dan menampilkan masjid terdekat; semua proses menghormati privasi Anda  
- **Internet**: Diperlukan hanya untuk mengambil konten dari API Amalanku (waktu shalat, doa, dzikir)  
- **Data Pribadi Tidak Dikirim**: Aplikasi tidak mengirim info pribadi Anda ke API  

### Yang Tidak Diakses:

- **Kamera**: Tidak diakses  
- **Kontak**: Tidak diakses  
- **Mikrofon**: Tidak diakses  
- **Telepon**: Tidak diakses  

**Catatan**: Semua pemrosesan data terjadi secara lokal di perangkat Anda. Tidak ada data yang dikirim ke server eksternal atau pihak ketiga.
