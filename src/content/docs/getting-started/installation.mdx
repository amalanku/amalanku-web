# Instalasi Amalanku

Ikuti langkah-langkah berikut untuk mulai menggunakan aplikasi:

## 1. Download Aplikasi
- Android: [Google Play Store](https://play.google.com/store/apps/details?id=com.flagodna.amalanku)  
- iOS: [App Store](https://apps.apple.com/app/idXXXXXXXXX)  

## 2. Buat Akun Lokal
- Masukkan **nama pengguna** Anda  
- Pilih **level amalan**: Pemula, Menengah, atau Mahir  

## 3. Pilih Amalan
- Tentukan amalan harian yang ingin dicatat  

## 4. Atur Reminder
- Aktifkan notifikasi sesuai jadwal harian Anda  

## 5. Mulai Tracking
- Semua progress akan **tersimpan otomatis di perangkat**
- Data aman dan dapat diekspor sebagai PDF jika diperlukan
