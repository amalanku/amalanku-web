## Get Started

Start building your dream website with Luna Landing today! Download our template and discover the freedom and flexibility it offers to create a unique and captivating website.

### Prerequisites

```sh
npm install -g pnpm
```

### Install dependencies

```sh
pnpm i
```

### Run the project

```sh
pnpm dev
```
